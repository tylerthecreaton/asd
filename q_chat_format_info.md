# Q-CHAT Questionnaire Format

## Overview

Q-CHAT (Quantitative Checklist for Autism in Toddlers) เป็นเครื่องมือคัดกรองออทิสติกสำหรับเด็กเล็ก (18-30 เดือน) ที่ประกอบด้วยคำถาม 10 ข้อ

## Scoring System

- แต่ละคำถามมีคะแนน 0-1 คะแนน
- คะแนนรวมสูงสุด = 10 คะแนน
- ระดับความเสี่ยง:
  - Low risk: 0-3 คะแนน
  - Medium risk: 4-6 คะแนน
  - High risk: 7-10 คะแนน

## Response Options

แต่ละคำถามมี 2 ตัวเลือกคำตอบ:

### คำถามแบบ Standard (ข้อ 1, 2, 4, 5, 6, 8, 10)

- "ไม่ใช่" (No) = 1 คะแนน (บ่งชี้ความเสี่ยง - เด็กไม่แสดงพฤติกรรมตามปกติ)
- "ใช่" (Yes) = 0 คะแนน (พัฒนาการปกติ)

### คำถามแบบ Reverse/Atypical (ข้อ 3, 7, 9)

- "ใช่" (Yes) = 1 คะแนน (บ่งชี้ความเสี่ยง - เด็กแสดงพฤติกรรมผิดปกติ)
- "ไม่ใช่" (No) = 0 คะแนน (พัฒนาการปกติ)

## Question Categories

คำถามแบ่งเป็นหมวดหมู่ต่างๆ เช่น:

- การสื่อสาร (Communication)
- ปฏิสัมพันธ์ทางสังคม (Social Interaction)
- พฤติกรรมซ้ำซ้อน (Repetitive Behaviors)
- การเล่น (Play)
- การพัฒนา (Development)

## Sample Questions (Thai Translation)

1. บุตรหลายของคุณชอบที่จะถูกตัวเองหรือไม่?
2. บุตรหลายของคุณสนใจคนอื่นหรือไม่?
3. บุตรหลายของคุณจะชี้ไปที่สิ่งที่ต้องการหรือไม่?
   ... (25 ข้อทั้งหมด)

## Implementation Notes

- ใช้ระบบคำตอบแบบ Binary (2 ตัวเลือก: ใช่/ไม่ใช่)
- คะแนนสูง = ความเสี่ยงสูง
- มี 2 รูปแบบการให้คะแนน:
  - **Standard (qchat_standard)**: ตอบ "ไม่ใช่" = 1 คะแนน (ข้อ 1, 2, 4, 5, 6, 8, 10)
  - **Reverse (qchat_reverse)**: ตอบ "ใช่" = 1 คะแนน (ข้อ 3, 7, 9)
