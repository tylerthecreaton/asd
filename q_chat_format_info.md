# Q-CHAT Questionnaire Format

## Overview
Q-CHAT (Quantitative Checklist for Autism in Toddlers) เป็นเครื่องมือคัดกรองออทิสติกสำหรับเด็กเล็ก (18-30 เดือน) ที่ประกอบด้วยคำถาม 25 ข้อ

## Scoring System
- แต่ละคำถามมีคะแนน 0-2 คะแนน
- คะแนนรวมสูงสุด = 50 คะแนน
- ระดับความเสี่ยง:
  - Low risk: 0-15 คะแนน
  - Medium risk: 16-30 คะแนน
  - High risk: 31-50 คะแนน

## Response Options
แต่ละคำถามมี 3 ตัวเลือกคำตอบ:
1. "ไม่เคย/น้อยมาก" (Never/Rarely) - 0 คะแนน
2. "บางครั้ง" (Sometimes) - 1 คะแนน  
3. "มักจะ/เสมอ" (Usually/Always) - 2 คะแนน

## Question Categories
คำถามแบ่งเป็นหมวดหมู่ต่างๆ เช่น:
- การสื่อสาร (Communication)
- ปฏิสัมพันธ์ทางสังคม (Social Interaction)
- พฤติกรรมซ้ำซ้อน (Repetitive Behaviors)
- การเล่น (Play)
- การพัฒนา (Development)

## Sample Questions (Thai Translation)
1. บุตรหลายของคุณชอบที่จะถูกตัวเองหรือไม่?
2. บุตรหลายของคุณสนใจคนอื่นหรือไม่?
3. บุตรหลายของคุณจะชี้ไปที่สิ่งที่ต้องการหรือไม่?
... (25 ข้อทั้งหมด)

## Implementation Notes
- ต้องการปรับเปลี่ยนโครงสร้างคำตอบจาก 2 ตัวเลือก (ปัจจุบัน) เป็น 3 ตัวเลือก
- ต้องปรับวิธีการคำนวณคะแนน
- ต้องปรับระดับความเสี่ยงให้เหมาะสมกับ Q-CHAT
